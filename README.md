# Lento Auto Repair

Currently in development, this repository will eventually become the home for a modern, responsive website for Lento Auto Repair in Schiller Park, IL. Being built with clean HTML, CSS, and vanilla JavaScript.

## About

Lento Auto Repair provides professional collision repair, diagnostics, and maintenance services with fair pricing and a 24-month warranty. This website showcases our services, location, and makes it easy for customers to get in touch.

## Features

- **Responsive Design** - Mobile-first approach that works on all devices
- **Services Showcase** - Detailed information about collision repair, diagnostics, and maintenance
- **Contact Form** - Easy appointment booking and free estimate requests
- **Location Map** - Embedded Google Map with address and contact information
- **Bilingual Support** - Content available in English and Polish
- **SEO Optimized** - Structured data (JSON-LD) and meta tags for better search visibility
- **Accessibility** - WCAG compliant with proper semantic HTML
- **Business Hours** - Clearly displayed operating hours (Mon–Sat, 10:00am–7:00pm)

## Getting Started

### Local Development

1. Clone the repository:
   ```bash
   git clone https://github.com/Jakemiki29/Lento-auto-repair.git
   cd Lento-auto-repair
   ```

2. Open `index.html` in your web browser or use a local server:
   ```bash
   python -m http.server 8000
   # or
   npx http-server
   ```

3. Visit `http://localhost:8000` to view the site

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Grid, Flexbox, and responsive design
- **JavaScript (Vanilla)** - No dependencies
- **Google Maps API** - Location display
- **Schema.org** - Structured data for SEO

## Deployment

This is a static website and can be deployed to:

- **GitHub Pages** - Free hosting directly from your repository
- **Netlify** - Simple drag-and-drop or Git integration
- **Vercel** - Fast, serverless deployment
- **Traditional Web Hosting** - Any host that supports static files

### Deploy to GitHub Pages

1. Push to GitHub
2. Go to repository Settings → Pages
3. Select main branch as source
4. Your site will be live at `https://yourusername.github.io/Lento-auto-repair`
